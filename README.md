# 💼 Сервіс пошуку роботи (Job Search Service)

Цей проєкт створено в межах навчальної практики як командну роботу для реалізації зручного сервісу взаємодії між кандидатами та роботодавцями.

Проєкт реалізовано з дотриманням стандартів оформлення коду, чистоти архітектури та зрозумілого іменування змінних і методів. Для уникнення конфліктів при злитті код розбито на декілька окремих файлів.

---

## 🚀 Основний функціонал проєкту

Програма підтримує дві основні ролі користувачів із відповідними можливостями:

### 👤 1. Кандидат (Шукач роботи)
* Перегляд повного списку доступних вакансій у системі.
* Пошук вакансій за ключовими словами, категоріями або заробітною платою.
* Створення профілю кандидата та відправка резюме (відгук на вакансію).
* Перегляд історії своїх відгуків на вакансії.

### 🏢 2. Роботодавець
* Реєстрація та управління карткою компанії.
* Створення, редагування та видалення вакансій компанії.
* Перегляд відгуків і резюме від кандидатів на власні вакансії.
